# is-react-instance

> Check if module is loaded in a reactjs instance.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i is-react-instance --save-dev
```

## Usage

```js
var isReact = require('is-react-instance');
isReact() // true or false
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
